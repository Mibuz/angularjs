O que é AngularJS?

O que é AngularJS?

É um framework client-side MVC/MVVM feito em JavaScript e mantida pela Google.

"AngularJS é o que o HTML teria sido se tivesse sido projetado para aplicações"

Quem fez? Quem usa? Onde e quando usar?

Surgiu em 2009 por Miško Hevery e Adam Abrons ambos funcionários do Google na época.

  • ...

Iniciando com AngularJS

Iniciando com AngularJS

Para usar o AngularJS, basta incluir como qualquer outro script. O mais importante é utilizar o atributo ng-app ou ng-app="myApp".

Iniciando com AngularJS 

      <html lang="pt-BR">
      <head>
          <title>My App</title>
          <meta charset="UTF-8">
          <link rel="stylesheet" href="screen.css">
      </head>

Iniciando com AngularJS 

      <html lang="pt-BR">
      <head>
          <title>My App</title>
          <meta charset="UTF-8">
          <link rel="stylesheet" href="screen.css">
          <script src="angular.js"></script>
      </head>

Iniciando com AngularJS 

      <html lang="pt-BR"ng-app>
      <head>
          <title>My App</title>
          <meta charset="UTF-8">
          <link rel="stylesheet" href="screen.css">
          <script src="angular.js"></script>
      </head>

MVC

Todos já deve ter ouvido falar sobre MVC, usado em várias linguagens.

  1. Model - Trabalha com a estrutura de dados.
  2. View - É o HTML
  3. Controller - Controle dos dados.
  4. ViewModel - Ajuda a lógica de negócio a partir da interface.

MVVM

      <div ng-app="myApp">
         <a href="" ng-click="toggle = !toggle">Toggle nav</a>
         <ul ng-show="toggle">
            <li>Link 1</li>
            <li>Link 2</li>
            <li>Link 3</li>
         </ul>
      </div>
